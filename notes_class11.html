Definitions

What happens if random variables are not independent?

  • Then the variance becomes the covariance
  • cov(x,y) = \(\sigma_{xy}^2 = E[(x-\mu_x)(y-\mu_y) = E[xy] - E[x]\cdot E[y]\)
  • For discrete random variable:
  • \(cov(x,y) = \sum_x\sum_y(x-\mu_x)(y-\mu_y)P_{xy}(x,y)\)
  • For continuous random variable:
  • \(cov(x,y) = \int_x\int_y(x-\mu_x)(y-\mu_y)f_{xy}(x,y)\)
