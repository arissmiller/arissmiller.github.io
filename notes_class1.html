Definitions:

Experiment:

any procedure that can be repeated an infinite number of times and that has a well defined set of possible outcomes(e.g. rolling a die, taking a measurement or reading)

Sample Outcome:

each of the potential outcomes of an experiment

Sample Space:

total number and kind of possible outcomes e.g. coin -> 1-6, die -> H/T

Event:

a designated collection of sample outcomes, The entire sample space and null set. an event has occurred if the outcome of the experiment is one of the members of the event

Simple Event:

an event with only one outcome e.g. rolling a die once and getting a 6

Compound Event:

event that consists of multiple events e.g. getting an odd number when a die is rolled

Intersection of events A and B:

Outcomes common to both A and B, written as \(A \cap B\)

Union of events A and B:

All outcomes in events A and B (no repeats), written as \(A \cup B\)

Compelement of event A:

Set of outcomes not included in the outcomes of the event, written as \(A^c\)

Mutually Exclusive:

No outcomes in common, i.e. \(A \cap B = \emptyset\)
