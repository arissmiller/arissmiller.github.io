Definitions

Joint Densities:

  • Interaction of multiple variables in a given system

  • For discrete random variables

    • Range: all possible values that x and y can take
    • Joint probability mass function:
      • \(h(x,y) = P(X=x, Y=y) = h(x_i, y_i) = P(X=x_i, Y=y_i)\)
      • \(h(x_i, y_i) \geq 0\)
      • \(\sum_{i=1}^{n} \sum_{j=1}^{m} h(x_i, y_i) = 1\)
  • For continuous random variables:
    • \(f_{x,y}(x, y) \geq 0\) for all x and y
    • \(\int_{-\infty}^{\infty}\int_{-infty}^{infty}f_{x,y}dxdy = 1\)
    • domain is defined to be where the joing pdf is non 0, assumed to be 0 else where ###Marginal Distrubtions:
  • Probability mass function of each random variable x and y
  • For the random variable x, \(f(x_i) = \sum_{j=1}^{m} h(x_i, y_j)\)

  • For the random variable y, \(f(y_i) = \sum_{i=1}^{n} h(x_i, y_j)\)

  • For continuous random variables:
    • \(f_x(x) = \int_{-\infty}^{\infty}f_{x,y}(x,y) dy\) for all x\
    • \(f_y(y) = \int_{-\infty}^{\infty} f(_{x,y}(x,y) dx\)for all y

Example 6: X and Y are random variables with joint pdf:

\(f_{xy}(x,y) = \begin{cases}cx^2y & \quad 0 \leq y \leq x \leq 1 \newline 0 & \quad otherwise\end{cases}\)

a. Find Rxy and show in the x-y plane

b. Find c

\(1 = \int_{-\infty}^{\infty}\int_{-\infty}^{\infty}f_{xy}(x,y)dxdy\)

\(1= \int_{0}^{1}\int{0}{x}cx^2y dydx = \int_0^1cx^2 dx\int_0^xy dy = \int_0^1cx^2(\frac{y^2}{2}\bigg|_0^x dx\)

\(1 = \int_0^1cx^2(\frac{x^2}{2}dx = \int_0^1 \frac{c}{2}x^4 dx = \frac{c}{10}x^5\bigg|_0^1 = 1\)

\(\frac{c}{10} = 1 \to c = 10\)

c. Find the marginal pdfs

\(R_x = R_y = [0, 1]\)

\(f_x(x) = \int_{-\infty}^{\infty}f_{xy}(x,y)dy = \int_0^x10x^2ydy = 10x^2\frac{y^2}{2}\bigg|_0^x\)

\(f_x(x) = \begin{cases}5x^4 & \quad 0 \leq x \leq 1 \newline 0 & \quad otherwise\end{cases}\)

\(f_y(y) = \int{-\infty}^{\infty}f_{xy}(x,y) dx = \int_y^110x^2ydx = 10y\frac{x^3}{3}\bigg|_y^1\)

\(f_y(y) = \begin{cases}\frac{10}{3}y(1-y^3) & \quad 0 \leq y \leq 1 \newline 0 & \quad otherwise\end{cases}\)

d. Find P(Y <= 2)

\(P(Y \leq 2) = \int_{-\infty}^{\infty}\int_0^2f_{xy}(x,y) dydx = \int_0^1\int_0^{\frac{x}{2}}10x^2y dydx = \int_0^1\frac{5}{4}x^4dx = \frac{1}{4}\)

e. Find P(Y <= x/4 | Y <=X/2)

\(\frac{P(y \leq \frac{x}{4}, y \leq \frac{x}{2}}{P(y \leq \frac{x}{2})}\)

\(4\int_0^1\int_0^{\frac{x}{4}}10x^2y dydx = 4 \int_0^1x^4dx = \frac{1}{4}\)
