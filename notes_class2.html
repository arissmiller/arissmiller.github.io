Class 2 Notes

Definitions

Probability function:

  • \(P(E) = n(E)/n(S)\)
  • Probability of any event e is a number between and including 0 and 1
  • The sum of all probabilities in the sample space is 1
  • If an event cannot occur its probability is 0
  • if an event is certain, the probability is 1
  • probability function is a mapping from an event to a number

Complementary events:

  • \(P(A^c) = 1 - P(A)\)

Mutually Exclusive Events:

  • if A and B are mutually exclusive, \(P(A \cup B) = P(A) + P(B)\)
  • if S has an infinite number events, and all are mutually exclusive, then the probability of the union of all events is the sum of all their individual probabilities

In General:

  • Probability of the null set is 0 (probability of no outcome is 0)
  • if A is a subset of B, then \( P(A) \leq P(B)\)
  • \(P(A \cup B) = P(A) + P(B) - P(A \cap B)\)

Three events that are not mutually exclusive:

  • \(P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(A \cap B) - P(A \cap C) - P (B \cap C) + P (A \cap B \cap C) \)

Examples

Probability that a card drawn at random from a deck of cards is either a 9 or a diamond:

\(P(9 \cup D) = \frac{4}{52} + \frac{13}{52} - \frac{1}{52} = \frac{16}{51} \approx 0.30\)

In a hostpital unit there are 8 nurses and 5 physicians, 7 nurses and 3 physicians are female. If a saff person is selected, find the probability that the subject is a nurse or a male.

\(P(Nurse \cup Male) = P(N) + P(M) - P(M \cup N)\)

Given one card is drawn from a deck. What is P(7 or 8 or 9):

\(P(7 \cup 8 \cup 9) = P(7) + P(8) + P(9)\)

\(P(S) + P(Q) + P(K) - P(S \cap Q) - P(S \cap K)\)

\( P(C) + P(F) - P(C \cap F)\)
