Definitions:

Factorial notation:

  • \(n! = n \cdot (n-1) \cdot (n-2) … 1\)

  • \(0! = 1\)

Rule 2: Counting Permutations

  • When all objects are distinct and order matters

  • an arrangement of n objects in a specified order

  • The number of permutations of length k that can be formed from a set of n distinct elements is \(nPk = \frac{n!}{(n-k)!}\) or the arrangments of n objectis in a specific order using k objects at a time

  • \(nPn = n!\)

Rule 3: Counting permutations when objects are not all distinct

Thm: number of ways n objects, such that \(n_1 =\) one kind

\(\frac{n!}{n_1! n_2! n_3! … n_k!}\)

multinomial coefficients

Approximation for when n is too large to do n!

\(n! \approx \sqrt{2\pi} \cdot n^{n + \frac{1}{2}} \cdot e^{-n}\)

\(\log_{10} n! \approx \log_{10}\sqrt{2\pi} + (n + \frac{1}{2})log_{10}n - n\log_{10}e\)

use the right hand side to get a number, and then raise 10 to the power of that number. This will approximate n!

Rule4: Counting combinations:

  • The concern is with the composition of a set of elements, not with any particular order of them.

  • k unordered elements is “a combination of size k”

  • Thm: # of ways to form combinations of size k from a set of n distinct objects with no repitition is \(nCk = {n \choose k} = \frac{n!}{k!(n-k)!}\)

  • \(nCk = \frac{nPk}{k!}\)

Circular Permutation:

non-linear arrangmement (objects not lined up in a row, e.g. seating at a circular table)

\((n - 1)!\) = number of ways to arrange n objects in a circle

Random Variable:

functions that associate numbers with some attribute of a sample outcome that is defined to be important.

Continuous random variable:

Random variable where outcome can be any real number in a given interval

Discrete random variable:

Random variable with a finite or countably infinte number of possible values

Binomial experiment:

  • There must be a fixed # of trials
  • Each trial can have only 2 outcomes
  • The outcomes of each trial must be independent of each other
  • The probability of success must remain the same for each trial
  • The outcomes of a binomial experiment and the corresponding probabiliities of these outcomes are called binomial distribution.

Notation for Binomial disbribution:

  • \(P(S)\) = probability of success
  • \(P(F)\) = probability of failure
  • p = numerical probability of success
  • q = numerical probability of failure
  • \(P(S) = p\)
  • \(P(F) = 1 - p = q\)
  • n = number of trials
  • x = number of successes in n trials

Binomial probability:

Random Variable \(X ~ B(n,p)\)

probability of getting k successes in n trials: \(P_r(k;n,p) = P_r(X=k) = {n \choose k}p^k(1-p)^{n-k}\)

Hypergeometric Distribution

In a hypergeometric distribution, the probability of success does not remain constant between trials, e.g. selecting without replacement

In this case we have a bucket with red and white chips. There are N total chips, r red chips and w white chips. k is the number or red chips selected. The probability that k = a number X is below:

\(P(X=k) = \frac{{r \choose k}{w \choose n-k}}{{N \choose n}}\)

Binomial expansion

A coins is tossed 3 times. Find the probability of getting exactly 2 Heads.

Binomial: # of trials = 3, Probability of success = (1/2)

\(P(X=2) = {3 \choose 2} \cdot (\frac{1}{2})^2 \cdot (\frac{1}{2})^{3-2} = \frac{3!}{2!(3-1)!} = \frac{3}{8}\)

A survery found that one out of 5 people has visited a doctor in any given month. If 10 people are selected at random, find the probability that exactly 3 will have visited a doctor last month

\(n = 10, X = 3, p = \frac{1}{5}, q = \frac{4}{5}\)

\(P(X=3) = {10 \choose 3} \cdot (\frac{1}{3})^3 \cdot (\frac{4}{5})^{10-3} \approx 0.201\)

A survey found that 30% of teenage consumer receive their spending money from part-time jobs. If 5 teenagers are selected at random, find the probabilty that at least 3 will have part time jobs.

\(n =5, p = 0.3, q = 0.7\)

\(P(X=3) = {5 \choose 3} \cdot 0.3^3 \cdot 0.7^{5-3} = \frac{5!}{3!2!} \approx 0.132\)

\(P(4) = {5 \choose 4} \cdot 0.3^4 \cdot 0.7^{5-4} \approx 0.028\)

\(P(5) = {5 \choose 5}\cdot 0.3^5 \cdot 0.7^{5-5} \approx 0.002\)

\(P(3) + P(4) + P(5) = 0.162\)
